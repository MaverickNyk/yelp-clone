# ExpressTemplate

ExpressTemplate is a nodejs template repo to build additional features on top of that.

## Installation

Use the node package manager npm to install dependencies.

```bash
npm install
```

## Deployment

Use the node deployment command to start the app.

```bash
node ./src/index.js
```
